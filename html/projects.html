Servant Engineering Our school brings together teams of engineers, from electrical, mechanical, biomedical and computer science to perform an act of service.

This project is helping a yogurt company in Uganda called Nguvu Dairy where we are developing a database that stores temperatures from wireless-thermometers that send its information to their locally stored website for their quality management system.

We are using a website for the team to use which is running off of Flask, CSS, and python with supporting ends using C and SQL.
Network Setup and Administration I have set up an Ubuntu and Windows server, that both have administrative tasks set on a separate windows client that can install, update and remove the clients software.

Perform scheduled scripts that create and maintain logs on user activity and server system storage. These servers have been set up with FTP, DNS and DCHP and will continually be developed to add more functionality.
